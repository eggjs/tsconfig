## Install

```shell
npm i --save-dev @eggjs/tsconfig
```

## Usage

```json
// tsconfig.json
{
  "extends": "@eggjs/tsconfig",
  // custom config
  "compilerOptions": {
    // override @eggjs/tsconfig options here
  }
}
```
